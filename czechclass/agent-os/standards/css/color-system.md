# Color System

Currently a purple theme. Treat as themeable — colors may change.

## Current Palette

| Role | Values |
|------|--------|
| Primary dark | `#4a148c` |
| Primary | `#9c27b0` |
| Primary light | `#ba68c8` |
| Primary bg | `#6a1b9a`, `#8e24aa` |
| Surface light | `#f3e5f5`, `#fce4ec`, `#ffe4e1`, `#fff0f5` |

## Feedback Colors (fixed)

| State | Background gradient | Text |
|-------|-------------------|------|
| Correct | `#90ee90` → `#98fb98` | `#006400` |
| Incorrect | `#ffb6c1` → `#ffa0a0` | `#4a148c` |
| Celebration | — | `#ffd700` (gold) |
| Progress | `#4caf50` → `#8bc34a` | — |

## Rules

- All backgrounds use gradients, not flat colors
- Text on dark backgrounds: `#ffffff`
- Text on light surfaces: primary dark (`#4a148c`)
- Keep feedback colors (green/pink/gold) consistent even if theme changes
